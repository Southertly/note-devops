centos: 7

iperf需要服务端与客户端一起工作, 类似于nc, 服务端监听端口, 客户端发送请求.

服务器端

```
iperf -s
```

- `-p`：指定服务器监听的端口或客户端连接的端口，默认为 5201

客户端

```
iperf -c 服务端ip -b 100M -t 60
```

向服务端ip的默认端口发送数据包, 带宽限制在 100Mb(注意是 Mbit, 与 MB 不同), 持续发送60s.
